During this project I got familiar with some of the python libraries such as Matplotlib, Sklearn.linear_model and Pandas. Before I had never worked with Matplotlib in python, so during this project I developed the basics on how to use Matplotlib. I use Matplotlib for plotting my data onto my graph using plt.

Another technology that I had never used before in python is called Sklearn.linear_model. This is a library for fitting a line to your data, to see if your data is linear or not. In order to be able to use the line in your code you must set the fit_intercept to be true, otherwise there will be no line to draw.

One more technology I used in this project was a library called pandas. This library is a very useful tool, particularly when working with csv files in excel. The main function I used in pandas was called, pd.read_csv(...). This line of code will read a csv file for you, you need to pass in a filename that you want to read.
