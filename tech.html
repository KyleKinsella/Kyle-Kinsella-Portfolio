Technologies that I have used or I will be using in the future

Below are some programming languages and a few technologies that are used for Machine Learning and Data Science.

Python: Python is one of the most popular and best programming languages for making machine learning application, this is because when you write a python program you can run it in parallel, this means that you can make your machine learning algorithms execute and learn very fast because they are running in parallel (a.k.a simultaneously).

R: R is a programming language that is mainly used for statistical computing and data visualisation.

Tensorflow: Tensorflow is a python library for creating machine learning models and you can run in any environment.

Anaconda: Anaconda is a python programming language distribution for scientific computing tasks such as: data science, machine learning applications, large scale data processing, predictive analysis.

Jupyter notebooks: A jupyter notebook is a collection of cells that can contain small bits of python code. This can be very good for learning new things such as Machine Learning and Data Science. Instead of having one big python program with lots of stuff that you don't understand, you can have a jupyter notebook with lots of small bits of code individually, then once you are comfortable you can combine them into one big python file if you wish.

Google Co.Labs: Google co labs is a service that provides jupyter notebooks, they also provide you with computer hardware such as, graphic processor unit (GPUs) and Tensor processing unit (TPU) This is a good service if you don't have a GPU in your laptop.

Database: In order for us to be able to use a Machine Learning Model, we need to be able to store the machine learning model and one of the best ways to do this is a database, the database that I think we might use is a SQL database.
